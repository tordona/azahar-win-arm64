# azahar-win-arm64
Unofficial Azahar builds for Windows on ARM (Qualcomm Snapdragon / ARM64)

* Qualcomm Snapdragon 8cx
* Qualcomm Snapdragon X
* Microsoft SQ Series

## Installation

### Windows
Azahar is available as both an installer and a zip archive.  

Download the latest release in your preferred format from the [Releases](https://github.com/tordona/azahar-win-arm64/releases/) page.  

If you are unsure of whether you want to use MSVC or MSYS2, use MSYS2.
